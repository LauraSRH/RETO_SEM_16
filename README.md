# RETO_SEM_16 Programa para consulta del clima.
Se dieron las especificaciones para el programa.
Se usaron funciones, try, if, elif, else para las distintas acciones del programa.
Se uso API_key de la página OpenWeather.
El programa se diseño con ayuda de la guía y revisión de errores con la IA del VSCode, 
pero dando indicaciones de algunas funciones, como poner el título, despedida y salida del programa para hacerlo más interactivo.
Ademas se tomaron en consideración errores como escribir mal las coordenadas, para su corrección, así como para las ciudades y siglas del país, en mayusculas con try y except.
